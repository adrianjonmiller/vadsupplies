=== WooCommerce Elavon VM Gateway ===
Author: skyverge
Tags: woocommerce
Requires at least: 3.8
Tested up to: 4.1
Requires WooCommerce at least: 2.1
Tested WooCommerce up to: 2.3

Adds the Elavon Virtual Merchant Gateway to your WooCommerce website. Requires an SSL certificate.

See http://docs.woothemes.com/document/elavon-vm-payment-gateway/ for full documentation.

== Installation ==

1. Upload the entire 'woocommerce-gateway-elavon' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
